# The Nix Package Manager

*by Jonathan Ringer and Tim Deherra*

This book assumes usage of the nix 3.0 (flakes) cli.

This can be enabled in nix 2.4+ by [following these instructions](https://nixos.wiki/wiki/Flakes#Enable_flakes).

This book is available on the web at [https://ekala-project.github.io/nix-book/](https://ekala-project.github.io/nix-book/).

For changes to the book, please see the [https://github.com/ekala-project/nix-book](https://github.com/ekala-project/nix-book).
